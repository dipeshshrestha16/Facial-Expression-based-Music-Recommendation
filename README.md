# Facial-Expression-based-Music-Recommendation
A Flask-based web application that detects facial emotions in real time using OpenCV and a deep learning model, and recommends music based on the detected emotion.

# Facial Expression-based Music Recommendation

This project detects facial expressions using a CNN model and recommends music based on the detected emotion.

## Features
- Facial expression recognition using deep learning (CNN)
- Emotion-based music recommendation using Spotify dataset
- Flask web application
- Real-time prediction from images

## Tech Stack
- Python
- TensorFlow / Keras
- Flask
- OpenCV
- Pandas, NumPy

## How to Run
```bash
pip install -r requirements.txt
python app.py
